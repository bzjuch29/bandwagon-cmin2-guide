## 先把概念搞清楚：CN2 和移动没有直接关系

CN2 是中国电信的网络，分两档：

- **CN2 GT（Global Transit）**：电信的中端线路，早年口碑不错，2019 年以后拥堵开始严重，现在已经下架了
- **CN2 GIA（Global Internet Access）**：电信最高端国际出口，稳定性好，晚高峰丢包率极低，走的是独立带宽

注意关键词：这是**电信**的线路。如果你是移动用户，访问走的是移动自己的骨干出去，不会自动跳到 CN2 GIA——除非服务器那头做了专门的路由设置，让移动流量走移动自己的专线。

所以"搬瓦工 CN2 移动能用吗"，这个问题的准确回答是：**早期 CN2 GT 套餐移动体验差，现在 CN2 GIA-E 套餐移动体验很好，因为接入了 CMIN2。**

## CMIN2 是什么，为什么移动用户要关心这个

CMIN2 全称是 China Mobile International Network 2，简单说就是移动版的高端跨境专线。对应 AS 号是 AS58807。

过去移动用户访问美国机房，走的是移动的普通 163 骨干（AS9808 或 AS4134 绕行），晚高峰经常出现丢包甚至断线。CMIN2 解决了这个问题——它是直连通道，不绕，晚高峰表现能和电信 CN2 GIA 持平。

搬瓦工在 DC6 和 DC9 机房正式接入 CMIN2 之后，官方通告里说得很直接：移动用户的延迟和吞吐量有显著改善。实际用下来也确实如此——同样是晚上十点用，接入 CMIN2 之前偶尔会感觉卡，接入之后基本没遇到过。

不卖关子了。**结论先给出来**：
